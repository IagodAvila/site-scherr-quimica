# 🧪 Scherr Química

Uma landing page institucional desenvolvida para apresentar os serviços da **Scherr Química**, destacando a empresa, seus diferenciais e formas de contato através de um design moderno, responsivo e focado na experiência do usuário.

🔗 **Demo:** https://iagodavila.github.io/site-scherr-quimica/

---

## 📋 Sobre o projeto

Este projeto foi desenvolvido simulando uma demanda real de um cliente, com o objetivo de criar uma presença digital profissional para uma empresa do setor químico.

O foco foi construir uma interface limpa, intuitiva e totalmente responsiva, transmitindo credibilidade e facilitando o acesso às principais informações da empresa.

---

## ✨ Funcionalidades

* Landing Page institucional
* Design moderno e responsivo
* Navegação fluida entre seções
* Layout otimizado para desktop, tablet e mobile
* Botões de chamada para ação (CTA)
* Seção de serviços
* Seção institucional (Sobre)
* Informações de contato
* Animações suaves para melhorar a experiência do usuário
